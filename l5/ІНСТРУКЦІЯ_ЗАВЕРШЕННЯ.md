## 🚀 Крок 1: Підготовка GitHub репозиторію

### 1.1. Вийдіть на GitHub та перейдіть до вашого репозиторію

1. **Відкрийте браузер** і перейдіть на https://github.com/YOUR_USERNAME/YOUR_REPO
2. **Натисніть на вкладку "Issues"**

### 1.2. Створіть новий Issue

1. **Натисніть кнопку "New Issue"** (зелена кнопка)
2. **Заповніть поля:**
   - **Title:** `Laboratory Work #5`
   - **Labels:** Виберіть `documentation`
   - **Assignees:** Виберіть себе (ваш GitHub username)
3. **Натисніть "Submit new issue"**

**📸 СФОТОГРАФУЙТЕ:** Екран з новим Issue

### 1.3. Змініть статус Issue на "In progress"

1. На сторінці Issue натисніть **Status** в правій частині
2. Виберіть **"In progress"**

**📸 СФОТОГРАФУЙТЕ:** Issue зі статусом "In progress"

### 1.4. Створіть нову Git-гілку

1. У правій частині сторінки Issue знайдіть розділ **"Development"**
2. Натисніть **"Create a branch"**
3. **Затвердіть назву гілки:** `laboratory-work-5`
4. Натисніть **"Create branch"**

**📸 СФОТОГРАФУЙТЕ:** Повідомлення про створення гілки
