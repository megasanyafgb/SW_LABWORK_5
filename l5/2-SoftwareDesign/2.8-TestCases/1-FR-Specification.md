# Деталізована специфікація функції встановити_посаду()

## Опис функції

Функція `встановити_посаду()` призначена для встановлення нової посади для працівника в системі управління персоналом.

## Вхідні параметри

### 1. номер_працівника - номер ідентифікатора працівника
- **Тип данних:** number(4)
- **Умова 1:** номер_працівника повинен бути більше 0
- **Умова 2:** номер_працівника не повинен перевищувати 9999
- **Умова 3:** номер_працівника повинен існувати в базі даних

### 2. нова_посада - назва нової посади
- **Тип даних:** varchar(20)
- **Умова 4:** нова_посада повинна починатися з великої літери
- **Умова 5:** довжина рядка нова_посада не менше 3 символів
- **Умова 6:** довжина рядка нова_посада не більше 20 символів
- **Умова 7:** нова_посада не повинна містити спеціальних символів (допускаються тільки букви та дефіс)

## Вихідний параметр (Return)

- **Значення = 1:** Посада успішно встановлена
- **Значення = -1:** Номер працівника не знайдений або невалідний
- **Значення = -2:** Нова посада не відповідає вимогам формату
- **Значення = -3:** Помилка доступу до бази даних

## Приклади вихідних значень

| Вхідні параметри | Очікуваний результат |
|------------------|---------------------|
| номер_працівника = 5, нова_посада = "Менеджер" | Результат = 1 |
| номер_працівника = 10000, нова_посада = "Менеджер" | Результат = -1 |
| номер_працівника = 5, нова_посада = "аналітик" | Результат = -2 |
| номер_працівника = 5, нова_посада = "MN" | Результат = -2 |
| номер_працівника = 5, нова_посада = "Менеджер@Продажу" | Результат = -2 |
